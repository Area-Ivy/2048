## 2048
基于cocos2dx的2048小游戏
#游戏功能
基础⽅块：2和4，随机出现在棋盘上的空位。
合成规则：相同数字的⽅块在滑动时若相撞则合成⼀个新的⽅块，数值为原⽅块之和。
权值计算：每合成⼀个新数字，玩家得分为新⽅块的数字值。
背景⾳乐：⽀持背景⾳乐，以营造舒适的游戏氛围
基本界⾯：包含开始界⾯、暂停及恢复功能、退出功能
恢复功能：⽀持中途退出时记录当前状态，下次进⼊继续上⼀次游戏或重新开始
排⾏榜  ：⽀持本地排⾏榜
⾳效    ：⽅块合成时提供⾳效

以色块的颜色代替数字，具体对应关系如下：
2   ：淡黄色（#F7D561）
4   ：浅绿色（#A6E867）
8   ：绿色（#57D49A）
16  ：青绿色（#13B5B1）
32  ：蓝色（#448ACA）
64  ：紫色（#C863EA）
128 ：粉红色（#E173B5）
256 ：鲜艳的粉红色（#EE648D）
512 ：橙色（#F39D4F）
1024：橙色（#F57C4E）
2048：鲜红色（#F64C14）
